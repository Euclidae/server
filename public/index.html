Welcome to the Rust HTTP Server!

This is a complete HTTP server implementation built with Rust, featuring:

🚀 Fast & Safe

Built with Rust's memory safety and performance in mind

🔒 Secure

Directory traversal attack prevention built-in

📁 Static Files

Serves HTML, CSS, JS, and other static content

🔧 Modular

Clean architecture with trait-based design

Server Information

  • Address: 127.0.0.1:8080
  • Methods: GET, POST
  • Status: Active

Test the Server

Try these curl commands to test different endpoints:

curl http://127.0.0.1:8080
curl http://127.0.0.1:8080/hello
curl -X POST http://127.0.0.1:8080 -d "test data"
